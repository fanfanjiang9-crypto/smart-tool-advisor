## ✨ 核心特性

### 🩺 场景诊断
不假设用户懂技术。用大白话提问，逐步缩小需求范围。支持四类人群：
- 内容创作者（选题调研 / 内容生产 / 多平台运营 / 数据复盘）
- 技术开发者（需求设计 / 编码调试 / 代码审查 / 知识管理）
- 学习研究者（信息筛选 / 深度理解 / 输出表达 / 笔记管理）
- 商务运营者（文档处理 / 数据分析 / 沟通协调 / 流程自动化）

### 📊 场景化五维评分
不是通用维度，每个分数都带"**对你而言……**"的解释：
- **场景匹配度**（30%）：这个工具解决你的核心痛点吗？
- **上手难度**（20%）：以你的技术水平，多久能跑通？
- **性价比**（20%）：在你的预算范围内，是不是最优选？
- **扩展性**（15%）：用熟了还能帮你做什么？
- **可持续性**（15%）：值得投入时间学它吗？

### 🔄 智能推荐引擎
- **有目标工具 + 评测结果好** → 进阶升级 + 互补搭配
- **有目标工具 + 评测结果差** → 3 个更匹配的替代工具
- **没有目标工具** → 直接配 3-5 个工具组合 + 启动路线图

## ❓ FAQ

**Q: 这个 Skill 需要额外的 API Key 吗？**
A: 不需要。它只用 Claude Code 自带的 WebSearch/WebFetch 做工具调研，不依赖任何外部 API。

**Q: 和 tool-evaluator 之类的通用评测工具有什么区别？**
A: 通用评测工具是"你说一个工具，我给它打分"。Smart Tool Advisor 是"我先问清楚你是谁，再告诉你该用什么"。一个是被动评测，一个是主动顾问。

**Q: 支持英文吗？**
A: 当前版本以中文为主，但 Skill 的提问和输出框架可以通过修改 SKILL.md 轻松适配其他语言。

**Q: 推荐的 AI 工具信息是最新的吗？**
A: Skill 会实时搜索最新信息，不依赖预设的工具库。AI 工具更新很快，静态推荐很快就会过时。
