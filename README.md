# 🤖 Smart Tool Advisor — AI 工具智能顾问

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/platform-Claude%20Code-orange.svg)](https://claude.ai/code)

> **先诊断，后开方。** 同一个"帮我推荐工具"，不同的人会得到完全不同的答案——因为先通过 5 轮结构化提问摸清你是谁、做什么、卡在哪，再给你配武器库。

---

## 💡 和通用评测工具的本质区别

| | 通用评测 | Smart Tool Advisor |
|---|---|---|
| 起点 | 直接评测工具 | **先问 5 轮问题了解你** |
| 打分 | 通用维度 | **每个维度都关联你的场景** |
| 推荐 | 可选 | **无论结果好坏，必出替代/互补推荐** |
| 没目标时 | 没办法 | **直接基于画像配武器库 + 启动路线图** |
| 开源/闭源 | 不标注 | **🟢🟡🔴 四色标签区分** |

## ✨ 核心特性

### 🩺 场景诊断
不假设用户懂技术。用大白话逐步缩小需求范围。支持四类人群：
- 📝 内容创作者（选题调研 / 内容生产 / 多平台运营 / **数据复盘**）
- 💻 技术开发者（需求设计 / 编码调试 / 代码审查 / 知识管理）
- 📚 学习研究者（信息筛选 / 深度理解 / 输出表达 / 笔记管理）
- 💼 商务运营者（文档处理 / 数据分析 / 沟通协调 / 流程自动化）

### 📊 场景化五维评分
每个分数都带"**对你而言……**"的解释：
- **场景匹配度**（30%）：这个工具解决你的核心痛点吗？
- **上手难度**（20%）：以你的技术水平，多久能跑通？
- **性价比**（20%）：在你的预算范围内，是不是最优选？
- **扩展性**（15%）：用熟了还能帮你做什么？
- **可持续性**（15%）：值得投入时间学它吗？

### 🏷️ 开源/闭源四色标签
每个推荐工具强制标注：
| 🟢 开源 | 🟡 半开源 | 🔴 闭源 | ⚪ 免费但闭源 |
|---|---|---|---|
| GitHub 公开，可自部署 | 核心闭源/SDK 开源 | 完全依赖厂商 | 不收费但代码不公开 |

如果用户要求开源，推荐引擎优先推 🟢，闭源工具只在明显更优时标注风险后推荐。

### 🔄 智能推荐引擎
- **有目标工具 + 评测 ≥ 3.5** → 进阶升级 + 互补搭配
- **有目标工具 + 评测 < 3.5** → 3 个更匹配的替代工具
- **没有目标工具** → 3-5 个工具组合 + 启动路线图

## ❓ FAQ

**Q: 需要额外 API Key 吗？**
A: 不需要。只用 Claude Code 自带的搜索能力做工具调研。

**Q: 和通用评测工具有什么区别？**
A: 通用评测是"你说工具，我打分"。本 Skill 是"我先问清楚你是谁，再告诉你该用什么"。一个被动评测，一个主动顾问。

**Q: 支持其他 AI 编程工具（Cursor、Codex 等）吗？**
A: 通过 skills.sh 安装会自动适配多平台格式。详见 [skills.sh](https://skills.sh)。

**Q: 推荐的工具信息是最新的吗？**
A: Skill 会实时搜索最新信息，不依赖静态工具库。

**Q: 开源偏好是怎么处理的？**
A: 诊断阶段询问开源偏好 → 评测时四色标签标注 → 推荐时优先匹配开源工具。确保偏好被尊重。
